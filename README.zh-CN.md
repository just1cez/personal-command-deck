# Personal Command Deck

Personal Command Deck 是一个本地优先的桌面应用，适合想把每天的重点、任务、项目、提醒、快捷入口和收工复盘放在一个安静空间里的人。

它不是团队管理后台，也不是复杂日历或全能效率系统。它更像一个个人执行桌面：打开，看清下一步，然后开始做。

## 打开后你会看到什么

- 今日专注：当前目标、专注计时、完成率、累计专注时间和临近提醒。
- Top 3 任务：今天最重要的三件事。
- 普通待办：不应该抢占 Top 3 位置的小任务。
- 项目下一步：每个项目只显示下一步具体动作，而不是模糊长期目标。
- 快捷入口：常用工具、文档、邮箱、日历、GitHub、AI 工具和自定义链接。
- 灵感暂存箱：临时想法先丢进去，之后再整理。
- 提醒与倒计时：账单、deadline、生日、面试、旅行等日期。
- 收工复盘：记录今天做了什么、卡在哪里、明天第一步是什么。

## 主要功能

- 本地保存数据，不需要账号。
- 专注计时，并记录项目级专注时间。
- Top 3、普通待办和项目支持手动排序。
- 每日名言和本地名言池管理。
- 支持定位或输入城市查询天气。
- 支持 JSON 备份导入和导出。
- 可选 AI 生成每日复盘。
- Windows 安装程序支持选择安装目录。

## 安装

有正式版本时，可以从 GitHub Releases 下载 Windows 安装包：

[Releases](https://github.com/just1cez/personal-command-deck/releases)

运行安装程序，并在安装向导中选择安装位置。

如果还没有 Release，这个仓库目前提供的是源码。开发者可以在本地构建安装包。

## 日常使用方式

1. 打开应用。
2. 先看“今日专注”区域。
3. 从主专注卡片或项目卡片开始一个专注计时。
4. 临时冒出来的想法先放进灵感暂存箱，不打断当前任务。
5. 安排当天节奏时查看提醒和倒计时。
6. 收工时填写复盘，并生成一段简短总结。
7. 如果在意本地数据，定期导出 JSON 备份。

## AI 复盘

AI 复盘是可选功能。不开启 AI 时，应用也可以正常使用。

启用后，你可以选择 OpenAI-compatible 提供商、API 地址、API Key 和模型。应用会自动根据当前任务、项目、暂存箱、提醒、专注时间和复盘输入生成提示词。

内置提供商预设：

- OpenAI
- DeepSeek
- Moonshot
- 自定义 OpenAI-compatible 地址

API Key 保存在本机应用运行时。它适合个人本地使用，但不是加密密码管理器。

## 数据与隐私

- 指挥台数据保存在本机 localStorage。
- 应用不需要登录。
- 应用没有托管后端。
- 使用天气功能时，会请求公开天气和定位相关 API。
- 只有在启用 AI 并主动生成总结时，应用才会把自动生成的复盘提示词发送给你配置的 AI 服务商。
- 你可以在应用内导出和导入本地 JSON 备份。

## 给开发者

安装依赖：

```bash
npm install
```

本地运行：

```bash
npm run dev
```

运行桌面开发版：

```bash
npm run dev:desktop
```

构建 Windows 安装包：

```bash
npm run dist:desktop
```

构建产物会生成到 `dist/` 和 `release/`，这两个目录已被 Git 忽略。
